Project plan

Title

Panda Land

Goal

The goal of this website is to educate anyone that are intrested in learning more about these adorable creatures

Target Audience

My target audience will range from kids, adults, and zoologists

Main Features

My site will contain tabs that lead to diffrent sections of the website, facts about the pandas such as habitat and health info, a bunch of panda pictures to help set the mood

Personality

The mood of the site will be calm, relaxing and simple

The mood will help people be more passionate and sympathetic to the pandas

Limiting Factors

Website will be clean, simple and aesthetically pleasing so viewers with expert or beginner level of web browsing will find it easy to navigate.

Development guide

Site structure

  • Root Directory
    • images directory – contains jpgs and gifs
    • styles directory – contains all of the style sheets
    • docs directory- contains all research done as document files
    • index.html – home page for site, site purpose and overview
    • Gallery.html - Gallery of pandas and information about them based off of research
    • facts.html - facts about the pandas that will range from habitat, history and other fun facts
    • Contact.html - Contact information such as email will be done here
    • Support.html - Info about how to support the pandas with links to various support sites

Styles guide

Background

Background is a tan-ish color

{background-color: #ededed}

Default font

Text is black, 14 pixels

p { font-size: 14px; color: #00000;}

Header

The header section is 235px tall, positioned along the top of each page with the logo centered in the frame. The background is White.

header {
height: 235px;
background-color: #fffff;
background-image: url("/images/panda.png");
background-repeat: no-repeat;
background-position: center;
background-size: cover;
}

Navigation

The main navigation is 120px tall, positioned just below the header with a background color of Grey and White font color.

nav { height: 120px; background-color: #a3a3a3; color: #ffffff; }

Images

Images will be over white background with a grey border around the container

figure { background-color: ffffff; border: 2px solid #333333; }

img { padding: 1em; }

Links

Links will be blue. On hover, they will have a light green background and a purple border.

a { color: #24a5e0 }

a:hover { background-color: #42f554; border 2px dotted #a428a8; }
